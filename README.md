>🔐 Password Strength Checker

A Python-based tool that evaluates password strength using string handling and conditional logic — built as part of the DecodeLabs Cybersecurity Internship (Project 1).

> 📋 About

This project checks whether a password is **Weak**, **Medium**, or **Strong** based on:
- Password length (minimum 8 characters)
- Presence of uppercase letters
- Presence of numbers
- Presence of special symbols

> 🚀 How to Run

1. Clone this repository
2. Make sure Python 3 is installed
3. Run the script:
4. Enter a password when prompted to see its strength result

> 🛠️ Skills Used

- Python string handling
- Conditional logic
- Basic security concepts (entropy, password validation)
